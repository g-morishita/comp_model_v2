# Examples

Start with [complete_workflow/README.md](complete_workflow/README.md).

`example/complete_workflow/` is the new structured learning path for this repository:

- how to define a task and choose a model
- how to set priors and sample parameters
- how to fit with MLE and Stan
- how to run model comparison
- how to compare recovered parameters
- how to run the same workflow for each public model via `--model <model_id>`

The older per-model directories are still here as low-level references, but the
recommended entry point is the numbered workflow in `complete_workflow/`.
